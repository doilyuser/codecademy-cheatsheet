Codecademy Cheatsheet

HTML Semantic Tags for Structure

These “structural” tags communicate the layout of a page.

Tag Use Description
<header> Website headers The header tag defines content that should be considered the introductory information of a page or section
<nav> Contains navigation links The navigation tag is used for navigation links. It can be nested within the <header> tag, but secondary navigation <nav> tags are also commonly used elsewhere on the page.
<main> Wraps all content This tag contains the main content (also called the body) of a page. There should be only one tag per page.
<article> Contains independent content The article tag defines content that could stand independently of the page or site it’s on. It does not necessarily mean a “blog post.” Think of it more as “an article of clothing”—a self-contained item that can be used in various contexts.
<section> Groups similar content Using <section> is a way of grouping nearby content of a similar theme. A section tag differs from an article tag. It isn’t necessarily self-contained, but it forms part of something else.
<aside> Additional content An aside element defines content that’s less important. It’s often used for sidebars—areas that add complementary but nonessential information.
<footer> Website footers You use <footer> at the bottom of a page. It usually includes contact information, copyright information, and some site navigation.

HTML Semantic Tags for Text

These “semantic” tags convey the semantic function of the text they contain.

Tag Use Description
<h1> Headings The H1 tag marks the top level heading. There’s usually only one H1 heading per page.
<h2> - <h6> Subheadings The subheadings of various levels of importance. There can be multiple headings of the same level on a single page.
<p> Paragraphs A standalone paragraph of text.
<a> Links Used to mark up a hyperlink from one page to another.
<ol> Ordered Lists A list of items that are displayed in a particular order, starting with bullet points. One <li> (list item) tag contains a single item in the list.
<ul> Unordered Lists A list of items that do not need to be displayed in a particular order, starting with ordinal numbers. One <li> (list item) tag contains a single item of the list.
<q> / <blockquote> Quotes A quotation of the text. Use <blockquote> for long, multi-line quotations and <q> for shorter, inline quotations.
<em> Italics Used for text that should be emphasized.
<strong> Bold Used for text that should be strongly emphasized.
<code> Code A block of computer code.
Note: We’ve only listed some of the most common semantic HTML tags. You can use many others—like <summary>, <time>, <address>, <video>, etc.—to make the content of your website easier to understand.
