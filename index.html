HTML Semantic Tags for Structure

These “structural” tags communicate the layout of a page.

Tag Use Description
<header> Website headers The header tag defines content that should be considered the introductory information of a page or section
<nav> Contains navigation links The navigation tag is used for navigation links. It can be nested within the <header> tag, but secondary navigation <nav> tags are also commonly used elsewhere on the page.
